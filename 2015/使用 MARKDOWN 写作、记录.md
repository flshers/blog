#### 使用 Markdown写作/笔记

> Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面或者PDF等

1. 语法简单
2. 纯文本
3. 可以流畅完成书写
4. 更多的开发者网站支持此格式（github/简书/segmentfault/ituring/gitbook/agroup.baidu.com)
